# crypto-stacker

暗号通貨ポートフォリオ追跡・分析ツール

## 機能

- ポートフォリオ追跡
- 価格監視
- パフォーマンス分析
- 投資戦略

## 使い方

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/crypto-stacker.git
cd crypto-stacker

# 依存関係をインストール
npm install

# アプリケーションを起動
npm start
```

## コントリビュート

プルリクエストは歓迎します。大きな変更の場合は、まずイシューを開いて議論してください。
